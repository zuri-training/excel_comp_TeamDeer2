Differentiate between two excel files.

Free, fast and Easy way to comparison. Upload two separate files and find the differences or similarities.

What file type is supported?

This tool can read the following file formats:

  • Excel 2007+ (XLS/XLSX/XLSM
  • Comma Separated Values (CSV)

How to compare files

  • Upload the two XLS files to be compared separately.
  • Choose the options you will like to view.
  • Press the "COMPARE" button.
  • View the highlighting differences between your files.

Is my data secured on the platform?

Yes. Result files can be saved for later or downloaded immediately.

Can I compare Formulars?

No. This comparing tools only does Excel and CSV value based files.

What devices can I use?

It works from all platforms including Windows, Mac, Android and iOS. No software installation is needed.
